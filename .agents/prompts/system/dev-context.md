# project: Antigravity Scribe

A VS Code extension that captures Antigravity IDE agent sessions
(via CDP) into Obsidian-native Markdown notes.

## Tech stack

- Language: TypeScript, compiled with tsc + bundled with esbuild → dist/extension.js
- Runtime: Node.js ≥ 18, VS Code API ^1.85.0
- Key dependency: turndown (HTML → Markdown conversion for agent responses)
- WebSocket: ws package for CDP communication
- No test framework yet


## Working rules

1. When a task requires to understand the project context, first read `repomix-output.xml`, instead of reading src files directly.
   - If `repomix-output.xml` has not been provided, ask for it before starting any significant task.
2. Prefer targeted edits over rewrites.
3. Do not modify `dist/` — generated by build.
